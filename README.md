## Featured Projects

### ☁️ SKY — in progress

A mini-PaaS deployment platform built for the African market — the "Railway / Render" experience, adapted to local realities and budgets.

* Git-to-deploy workflow with containerized builds
* Bring Your Own Server launch strategy: deploy on your own VPS
* Focus on affordability, latency and local payment methods
* Stack: Next.js · Node.js · Docker · PostgreSQL · Redis

### 🚀 Bubbles

A local video conferencing platform.

* Real-time peer-to-peer video/audio with WebRTC
* Signaling & presence handled over Socket.io
* Email verification with one-click magic links, themeable UI (light/dark)
* Stack: Next.js · Spring Boot · PostgreSQL · WebRTC · Docker

### 🎥 Yoom

A full-featured video meeting platform (Zoom-style), designed to run smoothly on a local network.

* Real-time video/audio over WebRTC with self-hosted coturn (TURN/STUN) relays
* Signaling & presence over Socket.io
* In-meeting features: chat, "raise hand", collaborators and numbered notification badges
* Authentication with local accounts plus OAuth2 (GitHub / Google)
* LAN-ready HTTPS setup (mkcert + Caddy reverse proxy) for testing across devices
* Stack: Next.js · WebRTC · Socket.io · OAuth2 · Caddy · Docker

### 🏛️ SecureGov

A cybersecurity scanning & monitoring platform targeting the public sector in Cameroon (SecureGov_CM).

* Dashboard to launch, browse and track security scans, with saved filters/sorting per user
* Sovereign-AI triage: Groq in the cloud for dev, local Ollama (Llama 3.2) for on-premise / air-gapped deployments
* Polished UI: shadcn/ui components, Framer Motion transitions, true-dark theme
* Containerized infrastructure with Docker Compose (app + AI profiles)
* Stack: Next.js · TypeScript · shadcn/ui · Framer Motion · Docker · Groq / Ollama

### 🔐 Authentication API

A production-grade REST API implementing JWT authentication, role-based authorization and secure user management.

* Stateless `SecurityConfig`, custom `JwtFilter`, `UserDetails` integration
* Role enum handling and clean token claim parsing
* Stack: Spring Boot · Spring Security · JJWT · PostgreSQL

### 📦 Stock Management API

A backend for inventory management (categories, products, stock levels) built with a clean, layered architecture.

* JWT authentication with refresh tokens, BCrypt hashing, stateless security
* Role-based access (USER / ADMIN): public reads, admin-only writes
* Records-based DTOs with Jakarta validation, global exception handling, entity mappers
* Stack: Spring Boot · Spring Security · JJWT · JPA / Hibernate · PostgreSQL · Gradle
